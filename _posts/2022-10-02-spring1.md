# Spring

## JAVA EE
dependency injection, transaction management의 기능을 제공하지만 어려움, 복잡도가 높음, 여러 대처상황이 힘듬

## Spring
단순하게 라이브러리만 쓰는 방식으로 깔끔하게 처리  
dependency injection을 같이 쉽게 구현할 수 있게 해줌  
JAVA EE의 영역을 대신하게 됨!
